Codelve

Branding, UX/UI design, Graphic design, Web design, Logo design, Illustrator, Figma

Codelve is a web application where you can learn programming languages from tutorial videos. It shows you not only how to code for beginners but also how you use them professionally for advanced users.

The only thing that was decided by the client in the first place was the purpose of creating this web service, so I worked on whole process of designing from branding to visual design.
