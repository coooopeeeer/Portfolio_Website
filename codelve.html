Codelve

Branding, User research, UX design, UI design, Visual design, Graphic design, Web design, Logo design, Illustrator, Figma

Codelve is a web application where you can learn programming languages from tutorial videos. It teaches you not only how to code for beginners but also how you use them professionally for advanced users.

The only thing which was decided by the client in the first place was the concept of this web service, so I worked on whole design process from branding to visual design.

Let me walk you through my design process...

Design purpose

The client wanted to create the web service for those who want to learn programming languages and it to be accessible anytime/anywhere the users want.
Considering the current situation which people spend more time on smartphone, the client determined the lessons to be videos so that the users can access whenever they have spare time, even from outside.

The goals

  • Conversion rate: 1.5%
  • Website speed: less than 3 seconds
  • Help users to easily find next lesson to take

Key Challenges

  • For both beginners and advanced users
  • Has log in function with personalized progress charts
  • Design should be customizable by developers
